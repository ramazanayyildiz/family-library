# 📚 Family Library App - Quick Start

## 🚀 Get Started in 5 Minutes

### 1. Create Supabase Account (1 min)
- Go to [supabase.com](https://supabase.com)
- Sign up (free, no credit card needed)

### 2. Create Project (2 mins)
- Click "New Project"
- Name: `family-library`
- Save your database password!

### 3. Setup Database (1 min)
- Go to SQL Editor in Supabase
- Copy `supabase-schema.sql` from this project
- Paste and Run

### 4. Get API Keys (1 min)
- Settings → API in Supabase
- Copy URL, anon key, and service role key
- Paste into `.env.local`

### 5. Run the App!
```bash
npm run dev
```

Open http://localhost:3000 and create your account!

---

## 📱 What You Get

✅ Multi-user family library
✅ ISBN scanner (uses phone camera on mobile)
✅ Reading tracker (Want to Read, Reading, Read)
✅ Personal shelves & tags
✅ Family activity feed
✅ PostgreSQL database (production-ready!)
✅ Free hosting on Vercel

## 🌐 Deploy to Production

```bash
# Install Vercel
npm install -g vercel

# Deploy
vercel
```

Add your Supabase keys when prompted. Done! 🎉
